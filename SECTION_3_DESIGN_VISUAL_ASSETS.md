# DELIVERABLE 2: COLOR SCHEME & BRANDING GUIDE

## What to Create

Create a simple document (Google Docs, Word, or text file) with color and font choices for the website.

### Part 1: Color Palette

Choose 4-5 colors that work well together:

```
PRIMARY COLOR (for buttons, headers, main accents)
Color Name: Green
Hex Code: #2E7D32
RGB Values: 46, 125, 50
Usage: Buttons, headers, main brand color

SECONDARY COLOR (for accents and highlights)
Color Name: Orange
Hex Code: #FFA500
RGB Values: 255, 165, 0
Usage: Hover effects, highlights, secondary buttons

TEXT COLOR (main text)
Color Name: Dark Gray
Hex Code: #333333
RGB Values: 51, 51, 51
Usage: Body text, paragraph text

BACKGROUND COLOR
Color Name: White
Hex Code: #FFFFFF
RGB Values: 255, 255, 255
Usage: Main page background

ACCENT BACKGROUND (for sections)
Color Name: Light Gray
Hex Code: #F5F5F5
RGB Values: 245, 245, 245
Usage: Section backgrounds, alternating rows
```

### Where to Get Color Ideas:
- **Coolors.co** - Generate color palettes (FREE)
- **Paletton.com** - Build color schemes
- **Google Fonts** - See colors used in design examples
- Think about what represents markets: green (fresh produce), earth tones, vibrant colors

### Part 2: Font Choices

Choose fonts from **Google Fonts** (free and accessible to everyone):

**For Headings (large text):**
- Roboto Bold
- Playfair Display
- Poppins Bold
- Lato Bold

**For Body Text (regular text):**
- Roboto Regular
- Open Sans
- Poppins Regular
- Lato Regular

**Example:**
```
HEADINGS: Roboto Bold, size 24-36 pixels
SUBHEADINGS: Roboto Bold, size 18-24 pixels
BODY TEXT: Roboto Regular, size 14-16 pixels
```

### Part 3: Style Buttons & Hover Effects

Describe how interactive elements should look:

```
PRIMARY BUTTONS (e.g., Search, View Details)
- Background: #2E7D32 (Green)
- Text: White
- Text Size: 14px
- Padding: 12px 24px
- Hover: Darker green (#1B5E20)
- Cursor: Pointer

SECONDARY BUTTONS
- Background: #FFA500 (Orange)
- Text: Dark gray (#333333)
- Hover: Darker orange (#E69500)
```

### Part 4: Spacing & Layout

Guide for spacing:
```
Page margins (sides): 20px on mobile, 40px on desktop
Space between sections: 30-40px
Space between items in list: 15-20px
Card padding (space inside boxes): 20px
```

## FREQUENTLY ASKED QUESTIONS

**Q: Do my mockups need to be perfect?**  
A: No! They just need to clearly show layout. Simple is better.

**Q: Can I use pictures of real markets as reference?**  
A: Yes! Use real market photos for inspiration, but download free images to use.

**Q: Do all icons need to be from the same source?**  
A: Preferably yes - it keeps them consistent. But okay if needed.

**Q: What if I can't find the perfect image?**  
A: Use the closest match. Developers can adjust colors in editing later.

**Q: Can I use copyrighted images?**  
A: No! Only use images clearly marked as free to use. Always check licensing.
